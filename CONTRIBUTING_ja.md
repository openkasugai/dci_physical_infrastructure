# dci_physical_infrastructure へのコントリビューション

このリポジトリは DCI2.0 / DHRC の一部として公開されます。

質問、設計に関する要望、アイデアなどは GitHub [Discussions](https://github.com/openkasugai/dci_physical_infrastructure/discussions) を使用してください。

## コントリビューション受付状況

現時点では、バグ報告を歓迎します。一方で、新機能の要望、設計変更の提案、および Pull Request の受付は、2026年9月末に予定している正式リリース後に開始する予定です。

それまでは、主にバグ報告を GitHub Issues でお寄せください。

## バグの報告・機能追加要望

GitHub [Issues](https://github.com/openkasugai/dci_physical_infrastructure/issues) を使用してください。

### バグ

- label には `bug` を付与してください
- issue の先頭に以下を記載してください
  - 環境
  - DCI2.0 / DHRC version
  - 対象 component name
  - 問題事象
  - 再現手順

## セキュリティ脆弱性

セキュリティ脆弱性を public issue や discussion で報告しないでください。
セキュリティ脆弱性を報告する場合は、GitHub の [Private vulnerability reporting](https://github.com/openkasugai/dci_physical_infrastructure/security/advisories) 機能を使用してください。

## 機微情報

Issue、Discussion、Pull Request、コミット、例、スクリーンショット、ログには、秘密情報、認証情報、実環境の IP アドレス、顧客情報、未公開構成情報、秘密鍵、トークン、未サニタイズのログを含めないでください。ログを共有する場合は、秘密情報、認証情報、実環境の IP アドレス、顧客情報、ホスト名、トークン、その他の機微情報を削除またはマスクしてください。
